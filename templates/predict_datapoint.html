The prediction feature requires a backend (Flask) route. GitHub Pages is a static host and cannot run the Flask app.

Options:

  • Run the app locally: see README for instructions (or run python application.py).
  • Host the Flask app on a dynamic host (Heroku, Azure, Render) and update this button to point to that URL.
  • Or implement a client-side prediction page that uses a hosted prediction API.
